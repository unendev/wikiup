# Technology Stack

## Frontend Stack
- **Framework**: Vue 3 (Composition API with `<script setup>`)
- **State Management**: Pinia
- **Routing**: Vue Router 4
- **HTTP Client**: Axios
- **Styling**: Tailwind CSS 3
- **UI Components**: Headless UI for Vue
- **Form Validation**: VeeValidate
- **Markdown Rendering**: Marked
- **Build Tool**: Vite
- **Language**: TypeScript

## Backend Stack
- **Framework**: Spring Boot 2.7.18
- **Language**: Java 8
- **Security**: Spring Security + JWT (jjwt 0.11.5)
- **Database**: MySQL 8.0 with Spring Data JPA
- **Connection Pool**: HikariCP
- **WebSocket**: Spring WebSocket
- **Reactive**: Spring WebFlux
- **ML/AI**: Deep Java Library (DJL) 0.27.0 with PyTorch
- **Vector DB**: ChromaDB (Java client 0.1.7)
- **Build Tool**: Maven
- **Utilities**: Lombok, Guava

## Architecture
- **Pattern**: RAG (Retrieval-Augmented Generation)
- **Communication**: REST API + WebSocket for streaming
- **Authentication**: JWT-based with role-based access control
- **Embedding Model**: Sentence-transformers multilingual model
- **LLM**: DeepSeek API integration
