The **Sweet Potato** is a [Vegetable](/wiki/Vegetable "Vegetable") [Food](/wiki/Food "Food") [item](/wiki/Item "Item") in the *[Shipwrecked](/wiki/Shipwrecked "Shipwrecked")* DLC. It can be found naturally in the [Meadow](/wiki/Meadow "Meadow") biome and can be eaten and cooked, much like the [Carrot](/wiki/Carrot "Carrot") in the original *[Don't Starve](/wiki/Don%27t_Starve "Don't Starve")*. Sweet Potatoes have exactly the same raw and cooked stats as carrots. Sweet Potatoes can be [Farmed](/wiki/Farming "Farming") by planting [Seeds](/wiki/Seeds "Seeds") for a 22.2% chance of growing or using Sweet Potato [Seeds](/wiki/Crop_Seeds "Crop Seeds"). It should be noted that Sweet Potatoes will only be available via farming in Worlds that are compatible with the *Shipwrecked* or *Hamlet* DLC.
