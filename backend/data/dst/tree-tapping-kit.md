The Tree Tapping Kit is an item added in [The Gorge](/wiki/The_Gorge "The Gorge"). Used to extract [sap](/wiki/Sap "Sap") from [sugarwood tree](/wiki/Sugarwood_Tree "Sugarwood Tree").

It can be purchased from [Pipton](/wiki/Pipton "Pipton") in Memorial Park for 3 [Old Coins](/wiki/The_Gorge "The Gorge") (for [Wes](/wiki/Wes "Wes"), two). The set contains 3 buckets for collecting [sap](/wiki/Sap "Sap").

You need to hang the bucket on one of the [sugarwood tree](/wiki/Sugarwood_Tree "Sugarwood Tree") in Memorial Park. Refills in 2 minutes.If the [sap](/wiki/Sap "Sap") is not collected in time, it will deteriorate, and the tree will be struck by the plague and will no longer produce sap.
