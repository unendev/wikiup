The **Weevole Carapace** is an [Item](/wiki/Item "Item") exclusive to the *[Hamlet](/wiki/Don%27t_Starve:_Hamlet "Don't Starve: Hamlet")* DLC. It is dropped by [Weevoles](/wiki/Weevole "Weevole") upon death.

Weevole Carapace can be used to make the [Weevole Mantle](/wiki/Weevole_Mantle "Weevole Mantle"). Although it cannot be eaten, it can be placed in the [Crock Pot](/wiki/Crock_Pot "Crock Pot") in order to make [Hard Shell Tacos](/wiki/Hard_Shell_Tacos "Hard Shell Tacos").
