The **Steamer Trunk** is a new object introduced in *[Shipwrecked](/wiki/Don%27t_Starve:_Shipwrecked "Don't Starve: Shipwrecked")* DLC. There are two variations of Steamer Trunks. First variation can be found in [Deep Ocean](/wiki/Ocean#Shipwrecked "Ocean") [Biome](/wiki/Biome "Biome"). There are 4 Steamer Trunks of this variation in the world, and each contains a few usable items and a [Trinket](/wiki/Trinkets "Trinkets") that can be traded to [Yaarctopus](/wiki/Yaarctopus "Yaarctopus") for [Dubloons](/wiki/Dubloons "Dubloons"). The contents of these are preset, and each trunk contains a different preset. The second variation is found in the [Ship Graveyard](/wiki/Ocean#Shipwrecked "Ocean") Biome. There are 4 to 8 Steamer Trunks of this variation in the world, and each contain a randomly generated set of common items in them.

Both variations function like a [Chest](/wiki/Chest "Chest") with 9 inventory slots and have a [Map](/wiki/Map "Map") icon, allowing player to easily find them again after sailing away. They can be used as storage on Ocean even after original contents are removed.

## Possible Contents[]

Steamer Trunks found in the Deep Ocean contain one of the following sets of items. All sets are available in different trunks around the world.

|  |
| --- |
| **Floral Shirt Umbrella Coconut ×2 Trinkets** |
| **Summer Frest Dumbrella Coconade ×2 Trinkets** |
| **Trinkets Snakeskin Hat Sewing Kit Life Jacket** |
| **Gold Nugget ×2 Straw Roll Honey Poultice Captain HatTropical Parasol Trinkets** |

For Steamer Trunks found in the Ship Graveyards, contents are randomly generated. Only a single item from each row from the following table can be found in a single chest, and items from rows with less than 100% chance may not be present at all. The durability or the amount of items vary as indicated, and all options are equally likely.

| Possible items | Chance to appear | Durability or Amount |
| --- | --- | --- |
| Log Suit Straw Hat | 25% | 15%-55% |
| Axe Machete Shovel Pickaxe | 100% | 15%-65% |
| Flint Dubloons | 50% | 1-3 |
| Coral Seaweed | 100% | 3-5 |

Note that [Seaweed](/wiki/Seaweed "Seaweed") in the trunks is created at world generation time, and if the trunk is not found quickly, it will turn to [Rot](/wiki/Rot "Rot").

## Placeholder Trivia[]

* A [Steamer Trunk](https://en.wikipedia.org/wiki/Steamer_trunk "wikipedia:Steamer trunk"), less commonly known as a Cabin Trunk, is a kind of travel luggage built to fit into various spaces (such as under the bed) in the cabin of steamships during the turn of the twentieth century. They are equivalent to the "carry-on luggage" in today's air travel.
* If a Steamer Trunk is spawned via [Console Commands](/wiki/Console "Console"), it will be empty.
