**Snake Oil** is an item exclusive to the *[Shipwrecked](/wiki/Don%27t_Starve:_Shipwrecked "Don't Starve: Shipwrecked")* DLC. It is a very rare joke drop from [Snakes](/wiki/Snake "Snake") and [Poison Snakes](/wiki/Poison_Snake "Poison Snake").

It can be quaffed like [Anti Venom](/wiki/Anti_Venom "Anti Venom"), but this doesn't remove [Poison](/wiki/Poison "Poison") or consume the Snake Oil. It can be removed by throwing it in a [Fire Pit](/wiki/Fire_Pit "Fire Pit") or [Chiminea](/wiki/Chiminea "Chiminea"), but it has no value as [Fuel](/wiki/Fuel "Fuel") either.

## Hamlet icon Hamlet[]

Snake Oil is also included in the *[Hamlet](/wiki/Hamlet "Hamlet")* DLC. It can be obtained rarely by killing [Vipers](/wiki/Viper "Viper").

## Placeholder Trivia[]

* "[Snake oil](https://en.wikipedia.org/wiki/Snake_oil "wikipedia:Snake oil")" is an expression that refers to health products with no verifiable benefits.
* There are 2 game files that prove that the intended use for Snake Oil is for it to have no use:
  + `poisonhealer.lua`: part of the 3rd line reads: "`--Used for snakeoil. We don't want it to actually do anything.`". "We" is referred to the developers.
  + `snakeoil.lua`: lines 23 to 27 read, which mean it is a fuel with a value of 0 and is a poison cure that does not cure at all:

```
   inst:AddComponent("fuel")
   inst.components.fuel.fuelvalue = 0

   inst:AddComponent("poisonhealer")
   inst.components.poisonhealer.enabled = false


```
