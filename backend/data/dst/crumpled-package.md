The **Crumpled Package** is an [Item](/wiki/Item "Item") exclusive to *[Don't Starve Together](/wiki/Don%27t_Starve_Together "Don't Starve Together")*, introduced in *[A New Reign](/wiki/A_New_Reign "A New Reign")*. It can be obtained by [fishing](/wiki/Fishing_Rod "Fishing Rod") in the [Lake](/wiki/Lake "Lake") at the [Oasis](/wiki/Set_Piece#Oasis "Set Piece"). Similar to [Bundled Supplies](/wiki/Bundled_Supplies "Bundled Supplies"), Crumpled Packages contain items, that will be dropped when opened.

Most packages contain [Trinkets](/wiki/Trinkets "Trinkets"), thus making them renewable. A special Trinket, the [Beach Toy](/wiki/Trinkets#Beach_Toy "Trinkets") can only be acquired by finding it inside a Crumpled Package. There is also an another special trinket called the red crown which is from the cult of the lamb crossover. However, it may also contain the rare [Blueprint](/wiki/Blueprint "Blueprint") for the [Desert Goggles](/wiki/Desert_Goggles "Desert Goggles"), as well as various other Blueprints. The chance for a player of getting the rare blueprints decreases after they have been learnt.

| Item | Chance (Before Desert Goggles learned) | Chance (After Desert Goggles learned) |
| --- | --- | --- |
| Melty Marbles | 16.7% | 18.2% |
| Gord's Knot | 16.7% | 18.2% |
| Hardened Rubber Bung | 16.7% | 18.2% |
| Mismatched Buttons | 16.7% | 18.2% |
| Potato Cup | 1.7% | 1.8% |
| Blueprint | 8.3% | 9.1% |
| Desert Goggles Blueprint | 21.7% | 1.8% |
| Beach Toy | 1.7% | 14.5% |

## Prototype Tips[]

* During [Winter's Feast](/wiki/Winter%27s_Feast "Winter's Feast") event, the 4 common trinkets each have a 50% chance to be replaced by [Festive Baubles](/wiki/Festive_Baubles "Festive Baubles") (plain or fancy), or [Festive Lights.](/wiki/Festive_Lights "Festive Lights")
* Its possible to save the game *before* opening Crumpled Packages, and if none of the items dropped are desired, one can rollback the save to before the Crumpled Packages were opened and try again.
  + It must be noted that the player must load a rollback from the server menu rather than the in-game options, or else repeatedly loading a rollback can cause the game to always load the previous one.
  + Its recommended to learn the Desert Goggles crafting recipe before doing this, to increase the drop rate of standard Blueprints
